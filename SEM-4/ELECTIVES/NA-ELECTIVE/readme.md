Syllabus
Completion requirements

Unit 0: Numerical Analysis Tools: Review of Metric spaces and Basic Topology, Norms, Convergence, Contraction Maps, Quantifying Numerical Errors. (Lectures: 01)

Unit 1: Numerical Differentiation and Integration: Finite Difference. Numerical Integration (Riemann sum, Trapezoidal and Simpsons rule). Monte Carlo. Applications. (Lectures: 01)

Unit 2: Numerical Interpolations: Lagrange Interpolation, Polynomial interpolation, Bezier Curves (Lectures: 02)

Unit 3: Numerical Linear Algebra: Algorithms for LU, QR, SVD, Sparse matrices, Iterative methods for sparse matrices, Tensor Decompositions. Applications to quantization and image processing. Applications for LLM compression (Lectures: 02)

Unit 4: Numerical Optimisation: Convex Sets, Convex Functions, Linear, Quadratic, Semi-definite, and Conic Program. Introduction to Min-Max problems. Dynamic Optimisation. Applications to planning, finance, image completion. (Lectures: 04)
