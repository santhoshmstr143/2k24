# SciKit Learn

# Installation - MacOSX

1. Ensure that `python` and `pip` is installed on your system.

Check it by running :

```python
python --version
```

Similarly for pip :

```python
pip --version
```

1. Create a Virtual Environment

```python
python -m venv myenv
```

3.  Activate the virtual environment

```python
source myenv/bin/activate
```

1. Install Scikit-Learn

```python
pip install -U scikit-learn
```

Check the installation by running :

```python
python -m pip show scikit-learn
```
